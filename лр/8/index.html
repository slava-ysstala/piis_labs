Основы адаптивной вёрстки

Как создавать сайты, которые работают везде

Приветствие. В этой презентации расскажем, что такое адаптивная вёрстка, зачем она нужна, какие технологии используются, и как начать верстать адаптивные сайты.

Что такое адаптивная вёрстка?

  • Подстраивается под ширину экрана
  • Удобна на телефоне, планшете, ноутбуке
Адаптивная вёрстка — это подход к веб-разработке, при котором интерфейс меняется в зависимости от устройства пользователя. Это значит — всё видно и удобно как на телефоне, так и на большом экране.

Зачем она нужна?

  • Половина трафика — с мобильных
  • Google понижает рейтинг неадаптивных сайтов
Адаптивная вёрстка — это уже не опция, а стандарт. Мобильные пользователи — это как минимум половина всех посетителей. Также поисковые системы отдают приоритет сайтам, которые адаптированы под разные устройства.

Media-запросы

@media позволяет применять CSS в зависимости от устройства


@media (max-width: 600px) {
  body {
    font-size: 14px;
  }
}
Media-запросы — основа адаптивной вёрстки. Они позволяют изменять стили при определённых условиях, например, когда ширина окна становится меньше 600 пикселей.

Flexbox — мощный инструмент для построения гибких макетов. Особенно хорошо работает при горизонтальном расположении элементов, а также при необходимости быстро менять порядок блоков.

Grid Layout

Современная система для сложных макетов


.container {
  display: grid;
  grid-template-columns: 1fr 2fr;
}
CSS Grid — это ещё более мощный инструмент, чем Flexbox. Он позволяет описывать двумерные макеты: строки и колонки. Подходит для создания сложных интерфейсов и таблиц.

Mobile-first подход

  • Сначала стили для телефонов
  • Потом — для более широких экранов
Подход mobile-first означает, что сначала мы пишем стили для самой маленькой ширины (обычно — телефоны), и далее добавляем стили для планшетов, ноутбуков и десктопов.

Что ещё поможет?

  • Фреймворки (Bootstrap, Tailwind)
  • Относительные единицы (em, %, vw)
  • Тестирование на реальных устройствах
Адаптивность не ограничивается CSS. Мы можем использовать фреймворки с готовыми компонентами, а также учитывать разные плотности экрана, пользоваться отладкой в DevTools и применять относительные единицы.

Итоги

Адаптивность = удобство + скорость + охват

Адаптивная вёрстка — это не просто тренд, а необходимость. Пользователи ждут, что сайт будет удобным и быстрым на любом устройстве. Это напрямую влияет на поведение, удержание и продажи.
